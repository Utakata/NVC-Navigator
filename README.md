# NVC プロセス・ナビゲーター

NVC（非暴力コミュニケーション / Nonviolent Communication）を実践するための包括的なWebアプリケーションです。マーシャル・ローゼンバーグ博士によって開発されたNVCの4つの要素（観察、感情、ニーズ、リクエスト）を様々な場面で活用できるよう、15種類のモードを提供します。

## 📋 特徴

- **15種類の実践モード**: さまざまな状況に対応した専用ガイド
- **語彙リファレンス**: 感情とニーズの包括的な語彙集
- **レスポンシブデザイン**: デスクトップ・モバイル両対応
- **印刷対応**: ワークシート形式での印刷が可能
- **日本語完全対応**: Shippori Mincho フォントによる読みやすい表示

## 🎯 15のモード

### 基礎学習
- **はじめに**: NVCの基本概念と4つの要素を学ぶ
- **観察の練習**: 観察と評価を区別する練習

### 日常の実践
- **対話モード**: 他者とのコミュニケーション準備
- **自己共感**: 自分自身の感情とニーズを探る
- **日常の実践**: 毎日の出来事を振り返る
- **感謝**: NVCの要素を使った感謝の表現

### 癒しと和解
- **癒しと和解**: 過去の痛みに対する自己共感と許し
- **敵イメージの解消**: 対立を超えて相互理解へ

### 組織・チーム
- **チーム会議**: 全員のニーズを考慮した会議進行
- **構造的協力**: 組織システムレベルでの変革設計
- **パフォーマンスレビュー**: ニーズベースのフィードバック

### 対立解決
- **仲介・調停**: 第三者としての仲介プロセス
- **修復的正義**: 罰ではなく修復と学びを通じた対応

### 深化と応用
- **ブレイクスルー**: 行き詰まりを突破する
- **社会変革**: NVCを社会に広げるビジョン

## 🛠 技術スタック

- **フレームワーク**: React 18
- **ビルドツール**: Vite 5
- **スタイリング**: Tailwind CSS 3
- **アイコン**: react-icons (Lucide)
- **フォント**: Google Fonts (Shippori Mincho)

## 📦 インストール

```bash
# リポジトリのクローン
git clone https://github.com/Utakata/NVC-Navigator.git
cd NVC-Navigator

# 依存関係のインストール
npm install
```

## 🚀 使用方法

### 開発サーバーの起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開いてアプリケーションにアクセスします。

### プロダクションビルド

```bash
npm run build
```

ビルドされたファイルは `dist/` ディレクトリに出力されます。

### プレビュー

```bash
npm run preview
```

プロダクションビルドをローカルでプレビューします。

## 🎨 主要コンポーネント

### NvcMasterApp
メインアプリケーションコンポーネント。ナビゲーション、モード管理、レイアウトを担当。

### ReferencePanel
感情とニーズの語彙リファレンスパネル。タブ切り替え機能付き。

### モードコンポーネント
各モード専用のコンポーネント（IntroductionMode、DialogueMode、SelfEmpathyModeなど）。

## 🖨 印刷機能

各モードには印刷対応スタイリングが適用されています。ブラウザの印刷機能を使用すると、ワークシート形式で出力できます。

## 🌐 NVCについて

NVC（Nonviolent Communication / 非暴力コミュニケーション）は、マーシャル・ローゼンバーグ博士により開発された、思いやりに基づくコミュニケーションと紛争解決の方法です。

### 4つの要素

1. **観察 (Observation)**: 評価や判断を交えず、具体的に何が起きたかを述べる
2. **感情 (Feelings)**: その観察に対して、自分がどう感じているかを表現する
3. **ニーズ (Needs)**: その感情の背後にある、満たされている/いないニーズを特定する
4. **リクエスト (Request)**: ニーズを満たすための、具体的で実行可能なお願いをする
